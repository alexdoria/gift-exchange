# Gift XCH
## Gift exchange web app built with python/django (3.6/3.2).

This is a web app to arrange a gift exchange between a group of friends. A mobile-first approach is intended by using [UIkit](https://getuikit.com/)

### The user can:
- Create an account with her/his email.
- Create a group.
- Create gifts.
- Assign gifts to selected groups.
- Sort out the members of each group to set gift matches.

### Next features:
- Invite friends via email. Invitations can be sent by now but the invited user will only see them if logged in. No email is sent.
- Upload custom gift picture. Replace current mockup.
- Show gift url and shorten it.
- Improve gift card UI, specially for group-set updating. It uses checkboxes and POST form, may be something with AJAX could work.
